Это справочная документация для {{<wiki "ModernWiki">}}.

## Как редактировать

Вы можете редактировать, нажав кнопку "Редактировать" в правом верхнем углу. Используйте синтаксис Markdown.

## Создание новой страницы

Используйте инструмент ниже для создания новой страницы:

{{< newpage-tool >}}

Структура каталога репозитория:

```
content/
├── docs/
│   └── категория/
│       └── имя_страницы.md
└── _index.md
```

Все файлы `.md` в каталоге `/content/docs/` будут отображаться как страницы.

### Загрузка изображений

Если вы хотите добавить изображения на страницу, преобразуйте страницу в каталог и поместите изображения в этот каталог.

Пример:

- У вас есть страница `content/docs/help/wiki.md`, и вы хотите добавить к ней изображения.
- Вам нужно изменить её на `content/docs/help/wiki/index.md`, а затем поместить изображения в каталог `content/docs/help/wiki/`.

## Ссылки на другие страницы

Можно использовать, например: `{{</*wiki "维基"*/>}}` => {{<wiki "维基">}}.

Если статья находится в категории, то: `{{</*wiki "категория/статья"*/>}}`

Если страница не существует, например `{{</*wiki "path/notExist.md"*/>}}` => {{<wiki "path/notExist.md">}}, она будет отображаться красным цветом.

{{< callout >}}
  Для прямой ссылки на URL используйте `[текст ссылки](URL)`
{{< /callout >}}

## Часто используемый синтаксис Markdown

Шпаргалка по основному синтаксису Markdown

| Элемент / Эффект               | Синтаксис                                 | Предварительный просмотр          |
| :----------------------------- | :---------------------------------------- | :-------------------------------- |
| **Заголовок (Heading)**        | `# H1` `## H2` `### H3`                   | /                                 |
| **Жирный текст (Bold)**        | `**Это жирный текст**` или `__Это жирный текст__` | **Это жирный текст**              |
| **Курсив (Italic)**            | `*Это текст курсивом*` или `_Это текст курсивом_` | _Это текст курсивом_              |
| **Жирный курсив (Bold & Italic)** | `***Это жирный текст курсивом***`         | **_Это жирный текст курсивом_**   |
| **Зачёркнутый текст (Strikethrough)** | `~~Это зачёркнутый текст~~`               | ~~Это зачёркнутый текст~~         |
| **Нумерованный список (Ordered List)** | `1. Первый пункт` `2. Второй пункт`       | /                                 |
| **Маркированный список (Unordered List)** | `- Пункт` `* Пункт`                       | /                                 |
| **Код (встроенный код)**       | `` Используйте функцию `printf()` ``      | Используйте функцию `printf()`    |
| **Блок кода (Code Block)**     | \`\`\` содержимое блока кода \`\`\`       | (отображается как подсвеченный блок кода) |
| **Разделитель (Horizontal Rule)** | `***` или `---`                           | /                                 |
| **Ссылка (Link)**              | `[текст ссылки](https://example.com)`     | [текст ссылки](https://example.com) |
| **Изображение (Image)**        | `![альтернативный текст](путь_к_изображению_или_URL)` | (отображается изображение)        |

## Другие расширения Markdown

### Математические формулы LaTeX

Используйте `\(...\)` для встроенных формул. Например: `\(\ce{H2O}\) — это вода` => \(\ce{H2O}\) — это вода.

Используйте `$$...$$` для формул в отдельной строке:

```latex
$$\mu_p=x^T\bm{\mu}$$
```

 $$\mu_p=x^T\bm{\mu}$$
