## كيفية التحرير

يمكنك التحرير بالنقر على زر "تحرير" في الزاوية اليمنى العليا، يرجى استخدام صيغة Markdown.

## إنشاء صفحة جديدة

يرجى استخدام الأداة أدناه لإنشاء صفحة جديدة:

{{< newpage-tool >}}

هيكل دليل المستودع:

```
content/
├── docs/
│   └── دليل_الفئة/
│       └── اسم_الصفحة.md
└── _index.md
```

جميع ملفات `.md` الموجودة في دليل `/content/docs/` سيتم عرضها كصفحات.

### رفع الصور

إذا كنت تريد إضافة صور إلى صفحة، يرجى تحويل الصفحة إلى دليل، ثم وضع الصور داخل هذا الدليل.

مثال:

- لديك صفحة `content/docs/help/wiki.md`، وتريد إضافة صور لها.
- تحتاج إلى تغييرها إلى `content/docs/help/wiki/index.md`، ثم وضع الصور في دليل `content/docs/help/wiki/`.

## الربط بصفحات أخرى

يمكنك استخدام مثلاً: `{{</*wiki "维基"*/>}}` => {{<wiki "维基">}}.

إذا كانت المقالة تحتوي على دليل فئة، فاستخدم: `{{</*wiki "الفئة/المقالة"*/>}}`

إذا كانت الصفحة غير موجودة، مثل `{{</*wiki "path/notExist.md"*/>}}` => {{<wiki "path/notExist.md">}}، فسيتم عرضها باللون الأحمر.

{{< callout >}}
  إذا كنت تريد الربط بعنوان URL مباشرة، يمكنك استخدام `[نص الرابط](URL)`
{{< /callout >}}

## صيغة Markdown الشائعة

ورقة غش سريعة لصيغة Markdown الأساسية

| العنصر / التأثير               | الصيغة                                     | معاينة التأثير                 |
| :---------------------------- | :--------------------------------------- | :------------------------------ |
| **العنوان (Heading)**         | `# H1` `## H2` `### H3`                  | /                               |
| **عريض (Bold)**               | `**هذا نص عريض**` أو `__هذا نص عريض__`   | **هذا نص عريض**                 |
| **مائل (Italic)**             | `*هذا نص مائل*` أو `_هذا نص مائل_`       | _هذا نص مائل_                   |
| **عريض ومائل (Bold & Italic)**| `***هذا نص عريض ومائل***`                | **_هذا نص عريض ومائل_**         |
| **خط في المنتصف (Strikethrough)** | `~~هذا نص مشطوب~~`                    | ~~هذا نص مشطوب~~                |
| **قائمة مرتبة (Ordered List)**| `1. العنصر الأول` `2. العنصر الثاني`     | /                               |
| **قائمة غير مرتبة (Unordered List)** | `- عنصر` `* عنصر`                    | /                               |
| **كود (كود مضمن)**            | `` استخدم دالة `printf()` ``             | استخدم دالة `printf()`          |
| **كتلة كود (Code Block)**     | \`\`\` محتوى كتلة الكود \`\`\`           | (تظهر ككتلة كود مظللة)          |
| **خط فاصل (Horizontal Rule)** | `***` أو `---`                           | /                               |
| **رابط (Link)**               | `[نص الرابط](https://example.com)`       | [نص الرابط](https://example.com)|
| **صورة (Image)**              | `![نص بديل](مسار الصورة أو URL)`         | (تظهر الصورة)                   |

## امتدادات Markdown الأخرى

### صيغ LaTeX الرياضية

استخدم `\(...\)` لتغليف الصيغ المضمنة في السطر. مثال: `\(\ce{H2O}\) هو الماء` => \(\ce{H2O}\) هو الماء.

استخدم `$$...$$` لتغليف الصيغ بين الأسطر:

```latex
$$\mu_p=x^T\bm{\mu}$$
```

 $$\mu_p=x^T\bm{\mu}$$
