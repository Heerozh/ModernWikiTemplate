これは {{<wiki "ModernWiki">}} のヘルプドキュメントです。

## 編集方法

右上の「編集」ボタンをクリックして編集できます。Markdown 構文を使用してください。

## 新規ページ作成

以下のツールを使用して新しいページを作成してください：

{{< newpage-tool >}}

リポジトリのディレクトリ構造：

```
content/
├── docs/
│   └── カテゴリディレクトリ/
│       └── ページ名.md
└── _index.md
```

`/content/docs/` ディレクトリ内のすべての `.md` ファイルはページとしてレンダリングされます。

### 画像のアップロード

ページに画像を追加する場合は、ページをディレクトリに変換し、そのディレクトリ内に画像を配置してください。

例：

- ページ `content/docs/help/wiki.md` があり、これに画像を追加したい場合。
- `content/docs/help/wiki/index.md` に変更し、画像を `content/docs/help/wiki/` ディレクトリ内に配置する必要があります。

## 他のページへのリンク

例えば、`{{</*wiki "维基"*/>}}` => {{<wiki "维基">}} のように使用できます。

記事がカテゴリディレクトリを持つ場合は、`{{</*wiki "カテゴリ/記事"*/>}}` のようにします。

ページが存在しない場合、例えば `{{</*wiki "path/notExist.md"*/>}}` => {{<wiki "path/notExist.md">}} は赤色で表示されます。

{{< callout >}}
  直接URLにリンクする場合は、`[リンクテキスト](URL)` を使用できます。
{{< /callout >}}

## よく使う Markdown 構文

基本的な Markdown 構文クイックリファレンス

| 要素 / 効果                     | 構文                                       | プレビュー効果                  |
| :------------------------------ | :----------------------------------------- | :------------------------------ |
| **見出し (Heading)**            | `# H1` `## H2` `### H3`                    | /                               |
| **太字 (Bold)**                 | `**これは太字テキスト**` または `__これは太字テキスト__` | **これは太字テキスト**          |
| **斜体 (Italic)**               | `*これは斜体テキスト*` または `_これは斜体テキスト_`   | _これは斜体テキスト_            |
| **太字斜体 (Bold & Italic)**    | `***これは太字斜体テキスト***`             | **_これは太字斜体テキスト_**    |
| **取り消し線 (Strikethrough)**  | `~~これは取り消しテキスト~~`               | ~~これは取り消しテキスト~~      |
| **番号付きリスト (Ordered List)** | `1. 最初の項目` `2. 2番目の項目`           | /                               |
| **箇条書きリスト (Unordered List)** | `- 項目` `* 項目`                          | /                               |
| **コード (インラインコード)**   | `` `printf()` 関数を使用する ``            | `printf()` 関数を使用する       |
| **コードブロック (Code Block)** | \`\`\` コードブロックの内容 \`\`\`         | (ハイライトされたコードブロックとして表示) |
| **水平線 (Horizontal Rule)**    | `***` または `---`                         | /                               |
| **リンク (Link)**               | `[リンクテキスト](https://example.com)`    | [リンクテキスト](https://example.com) |
| **画像 (Image)**                | `![代替テキスト](画像パスまたはURL)`       | (画像を表示)                    |

## その他の Markdown 拡張

### Latex 数式

インライン数式は `\(...\)` で囲みます。例：`\(\ce{H2O}\) は水` => \(\ce{H2O}\) は水。

ブロック数式は `$$...$$` で囲みます：

```latex
$$\mu_p=x^T\bm{\mu}$$
```

 $$\mu_p=x^T\bm{\mu}$$
